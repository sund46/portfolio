About Me

중학생 때부터 외부교육을 통하여 MS Office와 C 언어를 학습하며, 자격증 취득하거나 대회수상을 한 경험이 있습니다. 이후 컴퓨터전자공학을 전공하여 C, C ++, Python을 활용한 개발 프로젝트를 진행하면서 힘든 과정을 통하여 이루어 낸 결과물을 보고 느낀 성취감은 개발에 흥미를 느끼게 해준 계기가 되었습니다. 그래서 졸업 후 전문성을 향상하기 위하여 “자바 프레임워크 개발자 과정” 교육을 이수하였습니다. 교육을 통해 실무에 기반이 된 교육을 배우며 프로젝트를 통한 웹 UI 설계 및 기능구현을 하며 840시간 동안 개발 실력을 높이기 위해 노력했습니다. 그리고 Java 언어, Servlet, Spring Framework를 단계별로 이용하여 프로젝트를 진행하며 웹 개발기술의 이해도를 높였습니다.

저의 목표는 어떤 분야, 기술이든 개발할 수 있는 능력 있는 개발자가 되는 것이 목표입니다. 개발자의 덕목 중 하나는 시대에 뒤처지지 않고 개발 역량을 지속해서 발전해 나가는 것입니다. 그러므로 Java, JavaScript 등 업무와 관련된 프로그래밍 언어를 공부하고, 배우려는 자세로 업무에 빠르게 적응할 것입니다.
